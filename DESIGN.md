# 不倫募集掲示板サイト - 設計ドキュメント

## 要件まとめ

### ページ構成
1. **トップページ (index.php)**
   - 女性一覧は表示しない
   - SEOコンテンツセクション（不倫バレない方法、おすすめスポットなど）
   - 都道府県一覧セクション（47都道府県へのリンク）

2. **都道府県別ページ (prefecture-{slug}.php または page-{slug}.php)**
   - データベースから該当都道府県の女性一覧を表示
   - 都道府県別SEOコンテンツ（複数記事）
     - 不倫バレない方法（地域特化）
     - 不倫におすすめのスポット（地域特化）
     - その他関連コンテンツ

### デザインコンセプト
- **トンマナ**: 妖艶で大人っぽい雰囲気
- **カラースキーム**: ダークトーン（深い紫、ワインレッド、ゴールドアクセント）
- **フォント**: エレガントで洗練されたフォント
- **レイアウト**: 高級感のあるグリッドレイアウト

### データベース構造
- 既存テーブル: `wp_mama_gen`
- 追加必要カラム: `prefecture` (都道府県名)

## ファイル構成

```
hurin/
├── index.php              # トップページ（SEOコンテンツ + 地域一覧）
├── page-prefecture.php    # 都道府県別ページテンプレート
├── functions.php          # テーマ機能（カスタム投稿タイプ、SEOコンテンツ管理）

├── inc/
│   ├── prefecture-data.php    # 都道府県データ定義
│   └── seo-content.php        # SEOコンテンツ定義
└── assets/
    ├── css/
    │   └── prefecture.css     # 都道府県ページ専用スタイル
    └── images/
        └── (背景画像など)
```

## 都道府県ページURL構造

- `/prefecture/tokyo/` - 東京都
- `/prefecture/osaka/` - 大阪府
- `/prefecture/hokkaido/` - 北海道
など
