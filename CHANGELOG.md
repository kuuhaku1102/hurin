# 変更履歴

## Version 2.0 (2025-11-22)

### 新機能

#### 都道府県別ページ
- 47都道府県すべてに対応した専用ページを自動生成
- URL構造: `/prefecture/{slug}/` (例: `/prefecture/tokyo/`)
- 各都道府県ページに以下のコンテンツを配置:
  - 該当都道府県の女性一覧
  - 都道府県別SEOコンテンツ（4記事）
  - 他の都道府県へのリンク

#### トップページの刷新
- 女性一覧を削除（都道府県ページに移動）
- SEOコンテンツセクションを追加（4つの記事カード）
  - 不倫がバレない7つの鉄則
  - 既婚者が出会えるおすすめの場所
  - 理想の相手の見つけ方
  - トラブルを避けるための注意点
- 都道府県一覧セクションを追加（地域別グループ化）

#### SEOコンテンツの充実
- トップページに全国共通のSEOコンテンツを配置
- 都道府県ページに地域特化型SEOコンテンツを配置
  - {都道府県}で不倫がバレない方法
  - {都道府県}の不倫におすすめのスポット
  - {都道府県}で理想の相手と出会う方法
  - {都道府県}でトラブルを避けるための注意点

#### デザインの全面刷新
- **コンセプト**: 妖艶で大人っぽい雰囲気
- **カラースキーム**:
  - プライマリ: 深い紫 (#8B4789)
  - セカンダリ: ローズゴールド (#C17B7B)
  - アクセント: ゴールド (#D4AF37)
  - 背景: ダークトーン（グラデーション）
- **フォント**: 游明朝とゴシックの組み合わせ
- **エフェクト**: 
  - グラデーション背景
  - ホバーアニメーション
  - グローエフェクト
  - カードシャドウ

### 技術的な変更

#### データベース
- `wp_mama_gen`テーブルに`prefecture`カラムを追加
- 都道府県別のデータ取得機能を実装

#### ファイル構成
- `inc/prefecture-data.php`: 都道府県データ定義
- `page-prefecture.php`: 都道府県別ページテンプレート
- `assets/css/prefecture.css`: 都道府県ページ専用CSS
- `functions.php`: リライトルール、クエリ変数、テンプレート選択機能を追加

#### 機能追加
- カスタムリライトルールによる都道府県ページの実装
- 都道府県スラッグからデータを取得する関数
- 地域別に都道府県をグループ化する関数
- 都道府県別に女性データを取得する関数
