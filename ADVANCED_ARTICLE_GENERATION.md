# 高度な記事生成システム設計書

## 概要

Manusの高度な機能（Web検索、競合分析、AI文章生成）を活用し、SEOに特化した高品質なブログ記事を自動生成するシステム。

## 従来の問題点

- **テンプレート方式の限界**: 固定的な文章構造で、独自性に欠ける
- **SEO最適化不足**: キーワード調査や競合分析が不十分
- **文字数不足**: 2000-3000文字の充実した内容が作れない
- **最新情報の欠如**: トレンドや最新情報が反映されない

## 新しいアプローチ

### 1. 記事生成フロー

```
1. キーワード調査（Web検索）
   ↓
2. 競合記事分析（上位表示サイトの構成を調査）
   ↓
3. 記事構成の自動生成（見出し、キーワード配置）
   ↓
4. 本文の自動生成（AI文章生成）
   ↓
5. SEO最適化（メタディスクリプション、内部リンク）
   ↓
6. WordPress投稿
```

### 2. 技術構成

#### Pythonスクリプトによる記事生成

- **OpenAI API**: 高品質な文章生成
- **Web検索API**: キーワード調査と競合分析
- **WordPress REST API**: 自動投稿

#### 環境変数

```bash
OPENAI_API_KEY=（既に設定済み）
WP_URL=https://volitionmagazine.com
WP_USER=admin
WP_APP_PASSWORD=（既に設定済み）
```

### 3. 記事タイプ別の生成戦略

#### A. 不倫募集掲示板の活用法（毎日投稿）

**キーワード戦略**:
- メインKW: 不倫募集、不倫募集掲示板、セカンドパートナー
- サブKW: 不倫相手の見つけ方、既婚者の出会い、大人の関係

**記事構成**:
1. 導入（200-300文字）
2. 本論（1500-2000文字、H2×3-4個）
3. まとめ（200-300文字）

**生成プロセス**:
1. 「不倫募集掲示板 使い方」で検索
2. 上位3記事の見出し構成を分析
3. 独自の視点で記事を生成

#### B. 都道府県別の出会いスポット紹介（週3回）

**キーワード戦略**:
- メインKW: {都道府県} 不倫募集、{都道府県} 出会いスポット
- サブKW: {都道府県} デートスポット、{都道府県} 大人の出会い

**記事構成**:
1. 導入（都道府県の特徴）
2. おすすめスポット3-5箇所（各500文字）
3. 注意点とまとめ

**生成プロセス**:
1. 都道府県をランダムに選択
2. 「{都道府県} デートスポット」で検索
3. 実在するスポット情報を元に記事生成

#### C. 大人の関係のマナー（週1回）

**キーワード戦略**:
- メインKW: 不倫マナー、セカンドパートナー マナー
- サブKW: 大人の関係、不倫の心得

**記事構成**:
1. 導入（マナーの重要性）
2. 具体的なマナー5-7項目（各300文字）
3. まとめ

**生成プロセス**:
1. 「不倫 マナー」で検索
2. 上位記事から重要ポイントを抽出
3. 独自の視点で詳細に解説

### 4. SEO最適化要素

- **タイトル**: 32文字以内、キーワード含む
- **メタディスクリプション**: 120文字以内、魅力的な説明
- **見出し構造**: H2→H3の階層構造
- **キーワード密度**: 1-2%
- **内部リンク**: 関連記事へのリンク3-5個
- **画像**: アイキャッチ画像（AI生成も可能）

### 5. 実装方法

#### スクリプト構成

```
scripts/
├── generate-article.py      # メイン記事生成スクリプト
├── seo-optimizer.py          # SEO最適化処理
└── wordpress-poster.py       # WordPress投稿処理
```

#### GitHub Actions統合

```yaml
- name: Generate AI-powered article
  run: python3 scripts/generate-article.py --type usage --use-ai
  
- name: Post to WordPress
  run: python3 scripts/wordpress-poster.py
```
